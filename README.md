## Présentation

Ce projet est un jeu de quiz en ligne qui permet à plusieurs joueurs de :  

- Créer et gérer leurs propres quiz.  
- Jouer à des quiz existants via un **PIN unique**.  
- Suivre leur score et leurs performances.  
- Jouer en temps réel avec d’autres joueurs.  

---

## Fonctionnalités

- Authentification des utilisateurs (inscription / connexion).  
- Création, modification et suppression de quiz.  
- Système de multijoueur avec PIN pour rejoindre des quiz.  
- Gestion des points et du temps pour chaque question.  
- Interface web facile à utiliser.

## Installation

### Prérequis

- [Python 3.10+](https://www.python.org/downloads/)  
- [Node.js et npm](https://nodejs.org/)  
- [Docker](https://www.docker.com/get-started) 

### Pour jouer
Lancer docker-compose up --build dans le terminal
Aller sur http://localhost:8080/home.html pour lancer l'appli
